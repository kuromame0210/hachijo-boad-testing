# /pc-mock と /bright の表示項目比較

## /pc-mock に表示されている項目
- ヘッダー
  - ロゴ: `allo`
  - サブタイトル: `八丈島八丈島ライフロケット`
  - 日付: `YYYY年M月D日 (曜)`
  - 時刻: `HH:MM`
  - 操作: `設定` / `メニュー`
- アナログ風時計
  - リング / 島モチーフ / 針（時・分・秒）
  - デジタル時刻: `HH:MM`
- 生活情報
  - ごみ: `燃やせるゴミ` / `回収日`
  - 波浪: `2.5m`
  - 風速: `8m/s 北東`
- 交通情報（飛行機・橘丸）
  - ANA1891: 就航
  - ANA1893: 条件付き
  - ANA1895: 欠航
  - 橘丸 往路: 就航
  - 橘丸 復路: 条件付き
- 交通情報（アイランドシャトル）
  - 八→青: 就航
  - 青→八: 欠航
  - 八→御: 条件付き
  - 御→八: 就航
- 広告バナー
  - 見出し: `八丈島の夏、始まりました!`
  - サブ: `特産品フェア開催中 (広告)`
  - 画像枠: `イラスト` / `特産品`
- 下部ナビ
  - ホーム / マーケット / 投稿 / 設定

## /bright に表示されている項目
- ヘッダー
  - タイトル: `きょうの運航と天気`
  - 眉見出し: `Hachijo Island`
  - 更新時刻: `更新: {updatedAt}`
  - ナビリンク: `/bright` `/calm` `/test` `/test/preview` `/` `/weather`
  - アクション: `更新` / `詳細`
- 交通（カード2枚）
  - 東海汽船
    - ステータスバッジ: 欠航 / 条件付き / 遅延 / 運航 / 営業中 / 休業 / 情報なし
    - 便情報（八丈島行き / 八丈島発 / その他）
      - 時刻: `depPlanned` / `arrPlanned`
      - 便名/タイトル: `title`
      - 運航状況: `statusText`
      - 備考: `note`
  - 青ヶ島（船/ヘリ）
    - ステータスバッジ: 東海汽船と同様
    - 便情報（最大2件）: 時刻 / 便名 / 運航状況 / 備考
- 天気のサマリー
  - 波（底土）: 波高 `m` / 周期
  - 波（八重根）: 波高 `m` / 周期
  - 風: 風速 `m/s` / 突風 `m/s`
  - 台風: 台風名 or 台風番号 / 影響コメント
  - 天気の更新時刻: `天気の更新: {weatherUpdatedAt}`
- 参照元一覧（リンク）
  - 東海汽船 運航状況
  - うみそら便 青ヶ島
  - Open-Meteo 風
  - Open-Meteo 波
  - 気象庁 防災情報XML

## 比較（共通点と差分）
- 共通点
  - 交通情報を表示
  - 表示内容にステータス（就航/欠航/条件付き 等の概念）を含む
- /pc-mock のみ
  - 生活情報（ごみ/波浪/風速）
  - アナログ時計/デジタル時計
  - 広告バナー
  - 下部ナビ
- /bright のみ
  - 天気の詳細サマリー（波/風/台風）
  - 参照元リンクの一覧
  - 更新時刻の表示と更新アクション
